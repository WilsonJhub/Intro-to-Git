# Key notes to remember about Git

Git files have 3 main modes: 
- Committed
    - Information is securely stored in local database.
- Modified
    - File has been changed but **NOT** committed to the database.
- Staged
    - Flagged a fileâ€™s changed version to be committed in the next snapshot

### How to Clone a file:
 
   - $ git clone (Place link here)
  
  ## Check File Status:  
   - $ git status  

## Adding a file name:  
  - git add *filename*

## - Committing a File:  
  - git commit -m *"Insert change here"*

## Pushing Change to Origin
  - git push origin main
